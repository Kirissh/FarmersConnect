# Farmers Connect

A mobile application that enables farmers and buyers to connect directly, eliminating middlemen issues in produce distribution and marketing.

## Overview

Farmers Connect is a platform designed to bridge the gap between farmers and buyers by providing a direct marketplace. This eliminates traditional intermediaries, allowing farmers to get better prices for their produce while providing buyers with fresh, quality products at competitive rates.

## Features

- **Direct Farmer-Buyer Connection**: Connect directly without middlemen
- **Produce Marketplace**: Browse and list agricultural products
- **Real-time Communication**: Direct messaging between farmers and buyers
- **Quality Assurance**: Direct verification and quality control
- **Fair Pricing**: Transparent pricing without intermediary markups
- **User Profiles**: Dedicated profiles for farmers and buyers

## Technology Stack

## Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Kirissh/FarmersConnect.git
   cd FarmersConnect
   ```

2. **Open the Xcode project**
   ```bash
   open FarmersConnect.xcodeproj
   ```

3. **Install dependencies** (if using CocoaPods or SPM)
   ```bash
   # For CocoaPods
   pod install
   
   # For SPM, dependencies are managed in Xcode
   ```

4. **Build and Run**
   - Select the FarmersConnect scheme in Xcode
   - Choose your target device or simulator
   - Press `Cmd + R` to build and run

## Getting Started

### For Farmers

1. Create a farmer account
2. List your produce with details and pricing
3. Connect with interested buyers
4. Manage orders and delivery

### For Buyers

1. Create a buyer account
2. Browse available produce from farmers
3. Connect directly with farmers
4. Place orders and arrange pickup/delivery
